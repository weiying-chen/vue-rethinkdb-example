# Vue.js RethinkDB Example

Vue.js + Express + RethinkDB

## Installation

``` bash
# Install dependencies
npm install

# Install RethinkDB

http://rethinkdb.com/docs/install/
```

## Usage

```bash
# Serve RethinkDB at localhost:8080
rethinkdb

# Serve app with hot reload at localhost:8090
npm run dev
```
